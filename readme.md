# Wikidata Explorer

A simple application to search and visualize Wikidata entities with interactive knowledge graphs.

## Features

- Search for entities in Wikidata
- View detailed information about entities and their properties
- Visualize entity relationships in an interactive knowledge graph
- Display images directly in graph nodes
- Show relationship labels between entities

## Installation

1. Clone the repository:

```bash
git clone git@github.com:PratikHdhameliya/Wiki-data-explorer.git
cd Wiki-data-explorer
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Running Locally

To run the application locally:

```bash
python app.py
```

## Usage

1. **Search for Entities**:
   - Enter a search term in the search box (e.g., "Berlin", "Albert Einstein", "Weimar Republic")
   - Click "Search" to get a list of matching entities

2. **Explore Entity Details**:
   - Click on any search result to see detailed information
   - View the entity's properties and relationships

3. **Interact with the Knowledge Graph**:
   - The graph shows relationships between the selected entity and other entities
   - Nodes with images display the actual images
   - Hover over nodes to see more information
   - Drag nodes to reposition them
   - Zoom and pan to explore the graph
